# The Portfolio Gazette 🗞️

A premium, newspaper-themed portfolio website built with React and Vite. This design draws inspiration from vintage broadsheets, featuring paper textures, multi-column layouts, and classic typography.

## features

- **Vintage Broadside Aesthetic**: Immersive paper texture and serif typography.
- **Dynamic Columns**: Multi-column layouts for sections like "Field Dispatches" (Projects) and "Classifieds" (Skills).
- **Responsive Design**: Gracefully adapts to mobile and tablet viewports.
- **Functional Contact Form**: Integrated with Web3Forms for "Letters to the Editor".
- **Research Supplement**: Dedicated section for published academic papers.

## Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Custom broadsheet system)
- **Deployment**: [Live Demo](https://germanyportfolio.vercel.app/) (Vercel)

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
